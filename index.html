Virtual Drawing Board

Virtual Drawing Board allows us to write on the screen of laptops as if we are writing on a piece of paper and this doesn't even require the laptop to be touchscreen!
The main idea behind the project was that not everyone has iPads and most of the PCs today are not tochscreen. Due to this whenever we have to write using freehand like in online interviews or presentations we have to use mouse or mousepads which is not very convenient.
It uses a simple torch to write on the screen and can be very useful in presentations etc. Accurate and robust algorithm which can draw at high speeds without any considerable time lags. Pointer follows torch with very high precision almost instantly.
Coded many commands mapped to keys on keyboard which assist in drawing. No restriction on distance and angle between torch and computer for comfortable writing experience. Robust against almost all lighting conditions.
